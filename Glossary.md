
## 1. Workflow Context
*   **Control Rooms (CR):** Physical production environments (CR31, CR32, CR1A, etc.).
*   **Router Control:** Interaction with Evertz routers is sensitive; usually requested via backend services.
## 2. Glossary & Domain Lexicon
*   **Rundown:** The chronological list of stories/technical items in a news broadcast.
*   **iNews:** The Newsroom Computer System (NRCS) used to create rundowns.
*   **GPI / GPO:** General Purpose Input/Output. Hardware signals used for triggers.
*   **Cue:** Preparing a media item/graphic for playout.
*   **Take:** Executing the live transition of a cued item.
*   **Toolbox:** A collection of utility macros and branding elements for the operator.
*   **LSM:** Live Shot Management (The booking system).
*   **MTC:** Media Traffic Control (The team managing the lines).
*   **IFB:** Interruptible Foldback (Audio earpiece for talent).
*   **Viz/Vizrt:** Broadcast Graphics Engine. Graphics rendering and playback device 
*   **SIP:** Session Initiation Protocol (VoIP standard for audio routing).
*   Cuez - automation tool and vendor
* vPop - Producer driven production model used to Tier 3 or low cost productions 
* vPRC - virtual production control room, a control room using AWS and cloud infrastructure often used by tier 3 local production 
* Ross - Vendor who makes many production devices 
	* Overdrive - production automation tool used by local station 
	* Caprica - Multidevice communication interface 
	* Rosstalk - device communication protocol 
* Production Tiers 
	* Tier 1 -  The largest and most manual production offering used by a full control room manually controlling devices. Examples: Election Night, Nightly News, Today Show, SNL, Sports
	* Tier 2 - Local New, NBC News Now, Breaking News, Overnight coverage. An automated production operating at the highest available way. This tier is more capable than current local news and less Teir 1
	 - Tier 3- Editorial controls low cost 
 - Control Room Terminology 
	 - Control Room is a technology Stack 
	 - Gallery is the human interface 

## 3. **People and Roles** 
- TD - Technical Director 
- A1 - Audio Engineer / Audio Mixer / Production Mixer 
- A2 - Helps mic up and asissts the Audio Engineer 
- SM- Stage Manager 
- Jib - Camera operator who Operates a camera on a crane 
- V1 - Video Operator / Video Engineer / can also do "Robo"
- Robo - Robotic Camera Operator 
- TPM / TM - Technical Production Manager / Tech Manager. The manager in the control room who oversees the production and staffing. 
- VAP - Voice Activated Prompter
- Prompter - Teleprompter Operator 
- GFX Op - Graphics Operator who controls all on screen graphics and in studio monitors
- DPS / Tape - Digital Playback services / Video playback operators. 
- AD - associate director who helps time and count the show to and from break 
- Utility -  a technical associate who helps with physical tasks around a studio such as wrapping cables. Being an assist for a steadicam operator and other miscellaneous tasks related to live production
